# Stanford CS336: Language Modeling from Scratch

This repository tracks my work for Stanford CS336, **Language Modeling from
Scratch**. The course teaches language modeling by building the stack directly:
data handling, tokenization, Transformer model components, optimization,
training, systems work, scaling, data curation, and alignment.

The current CS336 offering is [Spring 2026](https://cs336.stanford.edu/). The
course site describes CS336 as an implementation-heavy class where students build
language models end to end, with minimal scaffolding.
